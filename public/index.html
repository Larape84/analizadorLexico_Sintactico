Sintasis GRANT y REVOKE SQL

instrucciones SQl para las sentencias GRANT y REVOKE

Las sentencias GRANT y REVOKE permiten a los administradores del sistema crear cuentas de usuario SQL y conceder y revocar derechos de esas cuentas

Puede utilizar la sentencia GRANT de SQL para otorgar privilegios SQL SELECT, UPDATE, INSERT, DELETE y otros en tablas o vistas, la cláusula WITH GRANT OPTION indica que JONES puede otorgar a otros

El comando REVOKE permite revocar privilegios de un usuario o un grupo o revocar la capacidad de un usuario o un grupo de conceder privilegios a otros usuarios.
